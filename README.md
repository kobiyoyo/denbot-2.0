## Description
This is a slack bot that performs basic arithmetic operations .

### Commands
- add 3+3 

## Built With
- Ruby 
- Slack-Ruby-Bot



## Getting started

> To get started with the app, clone the repo and then install the needed gems:


```
$ git clone 

$ bundle install

$ SLACK_API_TOKEN="Your api key here" rails s


```
